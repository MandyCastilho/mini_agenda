# 📅 Mini Agenda

Uma **Mini Agenda Virtual** simples, elegante e funcional, desenvolvida com **HTML, CSS e JavaScript puro**, focada em organização pessoal, produtividade e boa experiência do usuário. O projeto permite cadastrar compromissos, definir prioridades, filtrar por data e manter tudo salvo no navegador usando **LocalStorage**.

Este projeto foi pensado para ser leve, intuitivo e fácil de usar no dia a dia — perfeito tanto para estudos quanto para portfólio.

---

## ✨ Funcionalidades

* ➕ Adicionar compromissos com:

  * Data
  * Descrição
  * Prioridade (Alta, Média ou Baixa)
* 🗑️ Remover compromissos individualmente
* 🔍 Filtrar compromissos por data
* 📆 Ordenação automática por data
* 🎨 Indicação visual de prioridade por cores
* 💾 Persistência de dados com **LocalStorage**
* 🔔 Notificações do navegador ao adicionar novos compromissos
* 📱 Interface responsiva e moderna

---

## 🧠 Tecnologias Utilizadas

* **HTML5** — Estrutura semântica da aplicação
* **CSS3** — Estilização moderna, gradientes e responsividade
* **JavaScript (Vanilla)** — Lógica da aplicação e manipulação do DOM
* **LocalStorage** — Armazenamento local dos dados
* **Web Notifications API** — Notificações nativas do navegador

## ⚙️ Como Usar

1. Clone este repositório ou faça o download dos arquivos
2. Abra o arquivo `index.html` no navegador
3. Preencha:

   * A data do compromisso
   * A descrição
   * A prioridade
4. Clique em **Adicionar**
5. Use o filtro por data para visualizar compromissos específicos
6. Remova compromissos quando necessário

Todos os dados ficam salvos automaticamente no navegador.
